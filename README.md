# SSH Panel Bot

یک ربات تلگرام برای مدیریت و فروش اکانت‌های SSH با پنل وب ادمین.

## پیش‌نیازها
- سرور لینوکس (مثل Ubuntu 20.04 یا بالاتر)
- Python 3.8 یا بالاتر
- دسترسی به تلگرام و زرین‌پال
- دامنه با SSL/TLS (برای Webhook)
